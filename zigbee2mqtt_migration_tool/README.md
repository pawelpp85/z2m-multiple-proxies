# Zigbee2MQTT Migration tool

Migration helper for multiple Zigbee2MQTT instances. Stores IEEE -> name mappings, shows where devices live, and assists with removal/migration during pairing.

## Features
- Persistent mapping of IEEE addresses to device names
- Overview of device counts and health
- Recent activity feed aggregated from all instances
- Pairing status banner and migration actions

## Configuration
Provide the same instance URLs and optional tokens as the aggregated UI add-on.
